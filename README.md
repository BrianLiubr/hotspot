# Hotspot Hub

中文 AI / 财经 / 社会新闻热点聚合平台。

## MVP 目标

- 中文内容聚合
- 覆盖 AI / 财经 / 社会新闻
- 每 30 分钟自动刷新
- 支持手动刷新
- 网页展示总榜 / 最新 / 分类 / 事件详情

## 技术栈

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Jinja2
- APScheduler
- httpx / feedparser / BeautifulSoup / trafilatura
- Docker Compose

## 开发计划

1. 搭建 FastAPI 项目骨架
2. 接入 PostgreSQL 与 Alembic
3. 实现数据源抓取框架
4. 做标准化 / 分类 / 去重 / 排序
5. 完成页面与后台刷新
